# موقع الاستبيانات

موقع ويب بسيط لاستكمال الاستبيانات والحصول على نقاط.  
كل استبيان يكسب المستخدم 10 نقاط، والنقاط مخزنة محليًا على جهاز المستخدم.

## النشر على GitHub Pages
1. أنشئ مستودع جديد على GitHub.
2. ارفع جميع ملفات المشروع.
3. في إعدادات المستودع، فعل GitHub Pages من الفرع `main`.
4. سيتم الوصول للموقع من الرابط المعروض في GitHub Pages.
